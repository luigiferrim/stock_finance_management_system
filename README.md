## 📝 Descrição do Trabalho

O **Stock & Finance Management System** é uma aplicação web full-stack para gerenciamento de estoque e análise financeira. O sistema permite o controle de lotes de produtos com rastreamento de validade, visualização de métricas financeiras em tempo real e um sistema de auditoria que registra todas as ações críticas realizadas pelos usuários.
